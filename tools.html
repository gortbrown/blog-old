Tools

This is a set of tools that I self-host for anyone to use! I will add more as I start setting them up, so feel free to try them out!

Piped

Piped is an alternative privacy-focused frontend for YouTube.

ForgetMyPassword

ForgetMyPassword is a password manager that doesn't store passwords, but rather generates them for you when you need them.

MinimalPedia

MinimalPedia is a minimal, open source frontend for Wikipedia

Snowflake

Snowflake is a tool by the Tor Project that helps fight internet censorship. You can run a Snowflake proxy right from this site by flipping the switch below and leaving this tab open. When it turns green, that means someone is connected to your proxy and you are helping them evade censorship!
